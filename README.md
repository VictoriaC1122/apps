# VictoriaC1122 Apps Site

Standalone bilingual app showcase site for Victoria Cheng.

## Files

- `index.html`: single-page app showcase
- `styles.css`: visual style and responsive layout
- `script.js`: bilingual content and card rendering
- `assets/icons/`: local app icons copied from Xcode asset catalogs
